# Словарь для ИБ 

Модель CIA: Целостность-Конфиденциальность-Доступность

Гексада Паркера: Конфиденциальность-Целостность-Управляемость-Доступность-Полезность-Аутентичность

Модель STRIDE: Spoofing – Подмена, Tampering – Изменение данных, Repudiation– Отказ от ответственности, Information Disclosure- разглашение сведений, Denial of Service– отказ в обслуживании, Elevation of Privilege– захват привилегий

APT (Advanced Persistent Threat)- это хорошо организованная, тщательно спланированная кибератака, которая направлена на конкретную компанию или целую отрасль

0Day- это термин, обозначающий уязвимости, которые не были устранены каким-либо вендором до того как уязвимость стала публичной

Центр кибербезопасности (Security Operations Centre)– это центральное подразделение, которое обеспечивает организационные и технические вопросы безопасности

MSSP-Managed Security Service Provider-провайдер управляемых услуг информационной безопасности

MDR-Managed Detection and Response-услуга, предоставляемая компаниям для мониторинга, обнаружения и реагирования на киберугрозы в их IT-инфраструктуре

CERT-Computer Emergency Response Team-группа реагирования на компьютерные инциденты-CERT ориентирована на глобальные масштабы — работу в масштабах страны, региона или отрасли

CSIRT-Computer Security Incident Response Team-это команда реагирования на инциденты компьютерной безопасности- CSIRT обычно представляет собой небольшую формализованную группу внутри конкретного предприятия

Типы детектирующих правил: 

Common–правила работающие для всех клиентов и не имеющие в логике специфичных условий. 

Client-specific– правила работающие для определенных  клиентов.  Такие правила либо имеют такую же логику как  аналогичное common-правило, но со специфичными условиями или же правило может быть разработано с нуля основываясь на желаниях заказчика.
